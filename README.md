# Fullstack Challenge - Onfly 20231205

## Descrição

Este projeto consiste em um CRUD de despesas simples, onde é possível o usuário criar suas conta e fazer a gerencia de suas despesas.

## Tecnologias

### FrontEnd

- JavaScript
- TypeScript
- Vue

### BackEnd

- PHP
- Laravel

### Banco

- MySQL (HEROKO)

## Instruções

### FrontEnd

- cd ./frontend
- npm install
- quasar dev

### BackEnd

- cd ./backend
- composer install
- php composer serve

## Testes

- cd ./backend
- php composer test
